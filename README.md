# NailMuse Privacy Policy & Support

Last updated: July 2026

NailMuse is a mobile app that helps users save, organize, and plan nail inspiration before appointments.

## Information We Collect

NailMuse may collect:

- Email address for account login
- Uploaded nail inspiration photos
- Photo details such as category, color, finish, and notes
- Profile preferences such as favorite shape, length, style, colors, and notes
- Appointment board information

## How We Use Information

We use this information to:

- Create and manage user accounts
- Save nail inspiration photos
- Organize albums
- Save profile preferences
- Create appointment boards
- Help users share nail plans with their nail technician

## Data Storage

NailMuse uses Supabase for authentication, database storage, and image storage.

## Sharing Information

NailMuse does not sell personal information.

Users may choose to share their appointment board with a nail technician using the app’s share feature.

## Account Deletion

Users can request account deletion inside the app from the Profile screen.
